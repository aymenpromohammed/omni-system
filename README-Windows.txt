itqan soft - نظام نقطة المبيعات
===================================

خطوات التثبيت على Windows
--------------------------
1. ثبّت Node.js 20+ من: https://nodejs.org
2. فك ضغط هذا الملف في مجلد مثل: C:\itqan-pos
3. شغّل setup.bat (مرة واحدة فقط)
4. شغّل start.bat لتشغيل النظام
5. افتح المتصفح: http://localhost:5000

بيانات الدخول الافتراضية
-------------------------
المدير:  admin / admin123
الكاشير: cashier / cashier123

في حالة فشل setup.bat
---------------------
ثبّت Visual Studio Build Tools:
https://aka.ms/vs/17/release/vs_BuildTools.exe
اختر: Desktop development with C++
ثم اعد تشغيل setup.bat
